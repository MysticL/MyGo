# MyGo - 高效的文件搜索工具

一个专为macOS设计的快速文件搜索和管理系统，使用SwiftUI构建。

## ✨ 功能特性

### 🔍 智能搜索
- **实时搜索**：输入关键词即可实时显示搜索结果
- **高级查询语法**：支持复杂的搜索表达式
- **正则表达式**：支持正则表达式搜索
- **搜索历史**：自动保存搜索历史记录

### 📁 文件索引
- **智能索引**：自动为指定目录建立文件索引
- **实时监控**：监控文件系统变化，自动更新索引
- **性能优化**：高效的索引算法，支持大文件系统

### 🎯 高级筛选
- **文件类型筛选**：按文件扩展名过滤
- **大小筛选**：按文件大小范围过滤
- **日期筛选**：按修改时间过滤
- **路径匹配**：支持按路径搜索
- **大小写敏感**：可选择是否区分大小写

### 📋 文件操作
- **快速打开**：双击或回车快速打开文件
- **在Finder中显示**：快速定位文件在Finder中的位置
- **复制路径**：一键复制文件完整路径
- **文件移动**：支持拖拽和菜单移动文件
- **安全删除**：安全的文件删除操作

### ⚙️ 个性化设置
- **索引目录管理**：自定义要索引的目录
- **窗口大小记忆**：自动保存和恢复窗口大小
- **快捷键支持**：丰富的键盘快捷键
- **菜单集成**：集成到macOS菜单栏

## 🚀 快速开始

### 系统要求
- macOS 13.0+
- Xcode 15.0+

### 安装步骤
1. 克隆项目：
   ```bash
   git clone [repository-url]
   cd MyGo
   ```

2. 打开项目：
   ```bash
   open MyGo.xcodeproj
   ```

3. 构建并运行：
   - 在Xcode中选择目标设备
   - 点击运行按钮（⌘R）

### 首次使用
1. **设置索引目录**：
   - 打开应用后，点击菜单栏 "MyGo" → "设置..."
   - 添加要索引的目录（建议从常用目录开始）

2. **开始索引**：
   - 在设置中添加目录后，应用会自动开始索引
   - 或者手动点击菜单栏 "MyGo" → "重新索引"（⇧⌘R）

3. **开始搜索**：
   - 在搜索框中输入文件名或关键词
   - 使用筛选按钮进行高级筛选

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| ⌘, | 打开设置 |
| ⇧⌘R | 重新索引 |
| ↩ | 打开选中文件 |
| ⌘C | 复制文件路径 |
| ⌫ | 删除文件 |
| Space | 预览文件 |

## 🔧 高级用法

### 搜索语法
- **基本搜索**：直接输入文件名或关键词
- **路径搜索**：使用 `path:` 前缀搜索路径
- **类型搜索**：使用 `ext:` 前缀搜索文件类型
- **组合搜索**：支持多个条件组合

### 筛选器使用
- 点击搜索栏右侧的筛选按钮打开高级筛选面板
- 可以同时应用多个筛选条件
- 筛选器设置会自动保存

### 文件操作
- **右键菜单**：右键点击文件查看所有操作选项
- **拖拽操作**：支持将文件拖拽到其他应用
- **批量操作**：选择多个文件进行批量操作

## 📊 性能优化

- **索引优化**：智能跳过系统文件和隐藏文件
- **搜索优化**：使用数据库索引确保搜索速度
- **内存管理**：高效的内存使用和垃圾回收
- **后台处理**：索引操作在后台进行，不影响用户体验

## 🛠️ 开发信息

### 项目结构
```
MyGo/
├── Models/           # 数据模型
│   ├── FileItem.swift
│   └── SearchFilter.swift
├── Services/         # 业务逻辑服务
│   ├── DatabaseManager.swift
│   ├── FileIndexManager.swift
│   ├── FileOperationService.swift
│   ├── HotKeyManager.swift

│   ├── ContentView.swift
```

### 技术栈
- **UI框架**：SwiftUI
- **数据库**：SQLite（通过Core Data）
- **文件系统**：FileManager + FSEvents
- **并发**：Swift Concurrency
- **状态管理**：Combine + ObservableObject
