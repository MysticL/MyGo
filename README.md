## ✨ 功能特性

### 🔍 智能搜索

- **实时搜索**：输入关键词即可实时显示搜索结果，支持防抖优化
- **高级查询语法**：支持复杂的搜索表达式和查询解析
- **正则表达式**：支持正则表达式搜索模式
- **搜索历史**：自动保存最近 50 条搜索历史记录
- **路径白名单/黑名单**：支持自定义路径关键词列表，精确控制搜索范围

### 📁 文件索引

- **智能索引**：为指定目录建立高效的文件索引
- **实时监控**：监控文件系统变化，自动更新索引
- **性能优化**：高效的索引算法，支持大文件系统
- **索引管理**：支持添加、删除索引目录，手动触发重新索引

### 🎯 高级筛选

- **文件类型筛选**：按文件扩展名过滤（支持多选）
- **大小筛选**：按文件大小范围过滤（最小/最大）
- **日期筛选**：按创建/修改/访问时间过滤
- **路径匹配**：可选择是否在路径中搜索
- **大小写敏感**：可选择是否区分大小写
- **文件/文件夹过滤**：可单独筛选文件或文件夹
- **正则表达式模式**：筛选器支持正则表达式

### 📋 文件操作

- **快速打开**：双击或回车快速打开文件
- **在 Finder 中显示**：快速定位文件在 Finder 中的位置
- **复制路径**：一键复制文件完整路径
- **文件移动**：支持移动文件到指定目录
- **安全删除**：安全的文件删除操作，自动更新索引

### ⚙️ 个性化设置

- **索引目录管理**：自定义要索引的目录列表
- **路径关键词管理**：创建和管理路径白名单/黑名单
- **窗口大小记忆**：自动保存和恢复窗口大小
- **快捷键支持**：丰富的键盘快捷键【待更新】
- **权限管理**：自动检查和引导设置文件访问权限

## 🚀 快速开始

### 安装步骤

1. **克隆项目**：
   ```bash
   git clone https://github.com/MysticL/MyGo
   cd MyGo
   ```

2. **打开项目**：
   ```bash
   open MyGo.xcodeproj
   ```

3. **构建并运行**：
   - 在 Xcode 中选择目标设备（Mac）
   - 点击运行按钮（⌘R）或使用快捷键

### 首次使用

1. **权限设置**：
   - 首次启动时，应用会检查文件访问权限
   - 如果没有权限，会显示权限设置引导页面
   - 按照提示在系统设置中授予"完全磁盘访问权限"

2. **设置索引目录**：
   - 打开应用后，点击菜单栏 "MyGo" → "设置..."（⌘,）
   - 在"索引设置"标签页中添加要索引的目录
   - 建议从常用目录开始，避免索引过多文件

3. **开始索引**：
   - 添加目录后，应用会自动开始索引
   - 或手动点击菜单栏 "MyGo" → "重新索引"（⇧⌘R）
   - 索引过程中可以随时停止

4. **开始搜索**：
   - 在搜索框中输入文件名或关键词
   - 使用搜索栏右侧的筛选按钮进行高级筛选
   - 可选择路径白名单/黑名单进一步精确搜索

## 📖 使用指南

### 搜索语法

- **基本搜索**：直接输入文件名或关键词
  ```
  example.txt
  project
  ```

- **路径搜索**：使用 `path:` 前缀搜索路径
  ```
  path:Documents
  path:/Users/username
  ```

- **类型搜索**：使用 `ext:` 前缀搜索文件类型
  ```
  ext:swift
  ext:pdf
  ```

- **组合搜索**：支持多个条件组合
  ```
  project ext:swift path:src
  ```

### 筛选器使用

1. **打开筛选面板**：点击搜索栏右侧的筛选按钮（漏斗图标）

2. **设置筛选条件**：
   - **文件类型**：选择要搜索的文件扩展名（可多选）
   - **文件大小**：设置最小和最大文件大小
   - **日期范围**：选择日期类型（创建/修改/访问）和范围
   - **修饰符**：
     - 仅文件/仅文件夹
     - 区分大小写
     - 匹配路径
     - 使用正则表达式

3. **应用筛选**：筛选条件会自动应用，实时更新搜索结果

### 路径关键词列表

路径关键词列表用于创建白名单和黑名单，精确控制搜索范围：

- **白名单**：只搜索包含所有关键词的路径
- **黑名单**：排除包含任何关键词的路径

**使用示例**：
- 创建白名单 `["src", "swift"]`：只搜索路径中包含 "src" 和 "swift" 的文件
- 创建黑名单 `["node_modules", ".git"]`：排除路径中包含这些关键词的文件

### 文件操作

- **打开文件**：双击文件或选中后按回车（↩）
- **在 Finder 中显示**：右键菜单选择"在 Finder 中显示"
- **复制路径**：右键菜单选择"复制路径"（⌘C）
- **移动文件**：右键菜单选择"移动..."，选择目标目录
- **删除文件**：右键菜单选择"删除"（⌫），会从索引中移除

## 🛠️ 技术栈

- **UI 框架**：SwiftUI
- **数据库**：SQLite（通过 DatabaseManager）
- **文件系统**：FileManager + FSEvents（文件系统事件监控）
- **并发**：Swift Concurrency（async/await）
- **状态管理**：Combine + ObservableObject
- **日志系统**：自定义 Logger 服务

## 🔧 开发说明

### 开发环境设置

1. 确保安装了最新版本的 Xcode（15.0+）
2. 克隆项目并打开 `MyGo.xcodeproj`
3. 选择目标设备为 Mac
4. 运行项目（⌘R）

### 代码规范

- 遵循 Swift 官方编码规范
- 使用有意义的变量和函数名
- 添加必要的注释和文档
- 使用 Logger 服务记录调试信息

### 主要组件说明

- **DatabaseManager**：管理 SQLite 数据库，负责文件索引的存储和查询
- **FileIndexManager**：管理文件索引的创建、更新和监控
- **SearchService**：处理搜索逻辑，包括查询解析和结果过滤
- **SearchQueryParser**：解析搜索查询字符串，支持高级语法
- **FileOperationService**：处理文件操作（打开、移动、删除等）

### 日志系统

应用内置了日志系统，可以通过 `Logger.shared.log()` 记录不同级别的日志：
- `.debug`：调试信息
- `.info`：一般信息
- `.warning`：警告信息
- `.error`：错误信息
